项目实施流程

| 项目组长从Github仓库进行Fork,创建新的Branch进行项目开发 |
| ------------------------------------------------------- |
| 邀请小组成员进群                                        |
| 项目组长分配任务                                        |
| 小组成员下载翻译文档                                    |
| 小组成员查找平行文本，创建术语库                        |
| 小组成员完成翻译一稿，并提交翻译稿                      |
| 小组成员根据安排，对相应的翻译文稿在GitHub上进行审校    |
| 小组成员对翻译文稿进行修改，并提交翻译二稿              |
| 项目经理对提交的翻译稿进行汇总整理                      |
| 小组成员依据翻译文本进行Gitbook制作                     |
| 小组成员完成需要提交的所有文稿，并制作PPT               |
| 项目经理将最终成品仓库需通过Pull Request推送给原仓库    |
